# Prep Diplomacia (English)

Educational platform for preparing candidates for the **Uruguayan Foreign Service Entrance Examination**, directed by Carolina Techera.

## Stack

ASP.NET Core 8 MVC · ASP.NET Identity · Entity Framework Core (SQLite) · Stripe · Mailchimp · MailKit.

## Structure

Three layers following Clean Architecture:

- `PrepDiplomacia.Domain` — entities and business rules.
- `PrepDiplomacia.Infrastructure` — EF Core, Identity, Stripe, MailKit, Mailchimp.
- `PrepDiplomacia.Web` — public site, admin panel, student area.
